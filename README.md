# Projeto Sange - Trabalho de Conclusão de Curso

Projeto de software para a Clínica Fisioterapêutica Sange, criado para melhorar e facilitar o controle de pacientes para os funcionários da clinica.
Trabalho desenvolvido para a conclusão do curso de Análise e Desenvolvimento de Sistemas da FATEC Americana.
